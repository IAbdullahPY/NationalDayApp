## System Requirements
- Node.js version 18 or higher
- npm version 9 or higher
- Works on Linux, Windows, or macOS

## Dependencies
Install via npm:
- express
- body-parser
- cors
- sqlite3
- express-session

## Setup Instructions
1. Install Node.js from https://nodejs.org
2. Open terminal in project folder
3. Run:
   npm install
4. Start server:
   node server.js
